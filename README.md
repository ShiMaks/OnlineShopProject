# OnlineShop web project

Веб ресурс для заказа техники.
Создан самоподписной сертификат в keytool для протокола https.
Реализованный Функционал для гостя:
* просмотр доступных к покупке товаров в магазине (главная страница)
* выбор товара по категории
* просмотр информации о товаре
* регистрации (регистрация с уникальным логином и email)
* выбор языка интерфейса(русский, английский)
Реализованный функционал для пользователя:
* просмотр доступных к покупке товаров в магазине (главная страница)
* выбор товара по категории
* просмотр информации о товаре
* выбор языка интерфейса(русский, английский)
* Добавление товара(ов) в корзину
* Оформление заказа
* редактирование личных данных 
* просмотр своих заказов (текущих и завершенных) с подробной информацией о товарах входящих в заказ и их кол-ве
* изменение пароля
Реализованный функционал для администратора:
* просмотр всех товаров магазина
* просмотр категорий товара
* добавление товара и категории
* удаление товара/категории
* редактирование информации о товаре
* изменение названия категории
* просмотр списка заказов 
* сортировка заказов по статусу
* подтверждение заказа 
* просмотр списка пользователей
* просмотр заказов определенного пользователя
